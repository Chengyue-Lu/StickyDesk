# StickyDesk 使用说明

## 应用定位

StickyDesk 是一个常驻桌面的轻量便签工具，提供便签记录、未来任务、短时倒计时和活跃时长统计。

当前版本为迁移到 Tauri 架构后的首个正式版本：`v1.0.0`。

## 主要功能

- 便签：创建、编辑、排序、删除
- 未来任务：记录后续待办事项
- 短时倒计时：用于专注提醒，到点后触发系统通知
- 活跃时长：基于 Windows 系统空闲检测统计
- 窗口能力：置顶、透明度调整、非活跃自动淡出
- 外观设置：主题、界面缩放、便签排序方式

## 使用方法

### 便签

- 在主界面中直接创建和编辑便签
- 可按当前设置的排序方式查看内容

### 未来任务

- 在未来任务区域新增、编辑或删除条目
- 适合记录暂不处理但需要保留的事项

### 短时倒计时

- 启动倒计时后，应用会显示倒计时状态
- 倒计时结束后，当前条目会自动结束并触发 Windows 系统通知

### 活跃时长

- 应用会根据系统输入状态统计活跃时间
- 键盘输入和鼠标操作都会被视为活跃

## 数据存储

当前 Tauri 版本默认将数据保存到系统应用数据目录，而不是程序所在目录。

已保存的数据包括：

- 便签数据
- 未来任务数据
- 应用设置
